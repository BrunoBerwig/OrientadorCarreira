# Orientador de Carreiras

Este projeto é um orientador de carreiras que utiliza a API do Google Gemini para fornecer recomendações e insights personalizados. Desenvolvido com React, Vite, JavaScript e PostgreSQL.

## Tecnologias Utilizadas

- **Frontend:** React + Vite
- **Linguagem:** JavaScript
- **Banco de Dados:** PostgreSQL
- **API:** Google Gemini

## Desenvolvedor

Criado por **Bruno de Souza Berwig**.

## Como Executar o Projeto

1. Clone o repositório:
    ```bash
    git clone <URL_DO_REPOSITORIO>
    ```
2. Instale as dependências:
    ```bash
    npm install
    ```
3. Configure o banco de dados PostgreSQL.
4. Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```

## Funcionalidades

- Recomendações personalizadas de carreira.
- Insights baseados em dados fornecidos pela API do Google Gemini.
- Interface amigável e responsiva.
